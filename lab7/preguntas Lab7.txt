2.¿De que se encarga la funcion authorize() en el ActualizarAlbumREQUEST?
	Es autentificar el usuario y ademas comprobar de que el id del album le pertenece
	al usuario logeado actualmente.
	Finalmente esta funcion determina si esta petición o request podrá será
	actualizado o no, sólo si cumple la autentificaicon para este caso.

3.¿Que es lo que hace el codigo de la function postActualizarAlbum?
	Recibe un request; obtiene sólo el album con el id del request, se le asignan las
	atributos del album , como nombre, descripcion, que se obtienen del request,después
	se guarda el album, para finalmente redireccionarlo a /validado/albumes , con los 
	atributos de actualizado , que más adelante podrá servir para configurar un alert.

4.¿Qué se hizo con la function postActualizarAlbum en el controlador de Album?
